# Cà Phê Sấy Lạnh Việt Nam - Website Giới Thiệu Sản Phẩm

Website giới thiệu sản phẩm cà phê hòa tan sấy lạnh của Công ty TNHH Cà Phê Sấy Lạnh Việt Nam.

## Công nghệ

- **Next.js 16** - React framework
- **Tailwind CSS 4** - Styling
- **TypeScript** - Type safety

## Chạy dự án

```bash
# Cài đặt dependencies (đã cài)
npm install

# Chạy development server
npm run dev

# Build production
npm run build

# Chạy production
npm start
```

Mở [http://localhost:3000](http://localhost:3000) để xem website.

## Cấu trúc

- `src/app/` - App router, layout, trang chủ
- `src/components/` - Các component: Header, Hero, About, Product, Process, Certifications, Origin, FAQ, Contact, Footer

## Nội dung

Website bao gồm các phần:
- **Hero** - Giới thiệu sản phẩm chính
- **Giới thiệu** - Sứ mệnh, tầm nhìn, giá trị cốt lõi
- **Sản phẩm** - Điểm nổi bật, thành phần, đặc điểm
- **Quy trình** - 11 bước sản xuất
- **Chứng nhận** - HACCP, ISO 22000, FSSC 22000...
- **Nguồn gốc** - Vùng nguyên liệu Tây Nguyên
- **FAQ** - Câu hỏi thường gặp
- **Liên hệ** - Thông tin và form liên hệ
